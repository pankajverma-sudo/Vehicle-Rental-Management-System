# Vehicle Rental Management System

A simple Java OOP project that demonstrates:

- Inheritance
- Method Overriding
- Polymorphism
- super() Keyword

## Classes

- **Vehicle** (Base Class)
- **Car** (Subclass)
- **Bike** (Subclass)

## Features

- Calculate vehicle rental cost
- Different rent calculation for Car and Bike
- Runtime Polymorphism
- Constructor Chaining using `super()`
